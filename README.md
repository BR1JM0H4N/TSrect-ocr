## ✨ Overview

**Bulk OCR Extractor** is a beautiful, fully client-side Single Page Application (SPA) built with:

- 🧠 Tesseract.js for OCR
- 📄 PDF.js for PDF rendering
- 🎨 Modern responsive UI (Light/Dark mode)
- ⚡ Progressive processing (results appear instantly per page)
- 📲 Progressive Web App (PWA) support with offline caching

It allows you to upload multiple images or PDFs, extract text in English, Hindi, or both, and download results individually or combined.

No backend. No server uploads.  
Everything runs directly in your browser 🔐

---

## 🌟 Features

- 📂 Upload multiple images and PDFs at once
- 📑 Automatic PDF page detection
- 🔤 OCR Language options:
  - English (`eng`)
  - Hindi (`hin`)
  - English + Hindi (`eng+hin`)
- 📊 Live progress bar
- 🎛 OCR quality modes (Fast / Balanced / Accurate with preprocessing)
- 📌 Per-page preview & extracted text
- 📥 Download:
  - Individual page text
  - Combined full text
  - Searchable PDF for selected page/image (retains original look + text layer)
- 🛑 Stop OCR mid-process
- 🌗 Auto light/dark mode
- 📱 Fully responsive design
- 📦 Installable on desktop/mobile as a PWA

## 📸 How It Works

1. Upload images or PDFs
2. Select OCR language
3. Click **Start OCR**
4. Watch pages process progressively
5. Download results

Each page appears instantly once processed.

## 📦 Installation (Local Development)

Clone the repository:

```bash
git clone https://github.com/br1jm0h4n/TSrect-ocr.git
cd TSrect-ocr
```

Open `index.html` in your browser.

> ⚠ Make sure `tessdata` and `vendor` folders are properly included.
